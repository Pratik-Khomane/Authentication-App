# 🔐 Full Stack Authentication System

A simple full-stack authentication application built with **Spring Boot**, **Spring Security**, **JWT**, **React**, and **MySQL**.

## ✨ Features

* User Registration (Sign Up)
* User Login
* Password Encryption (BCrypt)
* JWT Authentication
* Role-Based Authorization
* RESTful APIs

## 🛠 Tech Stack

**Backend:** Java, Spring Boot, Spring Security, Spring Data JPA, MySQL

**Frontend:** React, React Router, Axios

## 🚀 Getting Started

1. Clone the repository.
2. Configure the MySQL database in `application.yaml`.
3. Run the Spring Boot backend.
4. Start the React frontend.
